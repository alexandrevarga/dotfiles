# 📂 DOTFILES REPOSITORY RULES

- **Direct Path Rule:** Verify that NO output file targets `~/.config` or `~/.local` directly. All write operations must target the `~/Projects/dotfiles/` directory structure.
- **Stow Verification:** Mentally dry-run GNU Stow symlink mapping before suggesting path changes (e.g., writing to `config-apps/ghostty/config` maps to `~/.config/ghostty/config`).
