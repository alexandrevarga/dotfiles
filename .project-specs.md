## 1. Zero-Trust Git Workflow (Agent Strict Directives)

This repository enforces a strict Pull Request-based workflow, even when GitHub's "Include administrators" branch protection is temporarily relaxed. Antigravity AI (Agy) **MUST ALWAYS** abide by the following hard-coded directives:

- **NEVER COMMIT DIRECTLY TO MAIN:** Agy is mathematically prohibited from executing `git push origin main` or any direct write operations to the `main` branch.
- **ISOLATED FEATURE BRANCHES:** Every logical unit of work MUST begin with the creation of a new branch (`git checkout -b feature/<name>`).
- **THE PULL REQUEST PIPELINE:** Code must ALWAYS be proposed via a Pull Request. 
  - Valid process: `gh pr create` -> `gh pr merge --squash --delete-branch`.
- **ENGLISH ONLY:** All commits, PR titles, PR bodies, and code documentation must be in English.

By reading this file, Agy acknowledges that the `main` branch is considered radioactive and untouchable by direct commands.
